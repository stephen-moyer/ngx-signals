createPager

Usage

General usage

Creates a pager object that can be used to load pages of data reactively using an effect. By default the page items for a given page are cached. This can be turned off using the pager options. 

    
    readonly page = signal({{ '{' }} pageNumber: 1, pageSize: 10 {{ '}' }});
    readonly pager = createPager(this.page, ({{ '{' }} pageNumber, pageSize {{ '}' }}) =>
      getPage(pageNumber, pageSize)
    );

page: {{ page().pageNumber }} (next, prev)

current page items: {{ pager().value?.page?.items | json }}

Notes

See observableEffectCache

The pagers value contains the following properties:

  • page: the current page
  • pages: all cached pages
  • allItems: items from all cached pages and current page combined
